# FutureSign registration demo

Статический сайт из двух страниц:

- **index.html** — современный главный экран с формой регистрации. Форма просит ФИО и номер телефона в формате `+7`.
- **entries.html** — "сломанная" страница списка заявок, отображающая всех зарегистрировавшихся (последний сверху).

Данные сохраняются в `localStorage`, поэтому они остаются в рамках одного браузера. Чтобы посмотреть работу:

1. Откройте `index.html` в браузере.
2. Заполните форму и нажмите «Зарегистрироваться».
3. Вас перенаправит на `entries.html`, где появятся все отправленные заявки.
